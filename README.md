# AI-FAQ--CHATBOT

## Overview
The AI FAQ Chatbot is an NLP-based chatbot developed using Python and Streamlit. It answers user questions by matching them with a predefined FAQ dataset using TF-IDF vectorization and Cosine Similarity techniques.

## Features
- Interactive chatbot interface
- FAQ-based question answering
- TF-IDF text vectorization
- Cosine Similarity matching
- Confidence score display
- Smart fallback response for unknown questions
- Professional Streamlit UI

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## Project Workflow
1. User enters a question.
2. Question is converted into TF-IDF vectors.
3. Similarity is calculated with FAQ questions.
4. Most relevant answer is identified.
5. Answer is displayed to the user.
